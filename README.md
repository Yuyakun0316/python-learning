# Python学習用リポジトリ

Pythonの習得を目指して学習したコードを記録しています。

## 📚 学習ログ
- Section 1: 基本構文（変数、データ型、演算子、f文字列）

- Section 2: 条件分岐と入出力
    - 真偽値 (`True/False`)、論理演算子 (`and/or/not`)
    - `if`, `elif`, `else` による分岐
    - `input()` 関数によるユーザー入力と型変換 (`int()`)
    - 成果物: お買い物アプリ
- Section 3: リストとループ (List, Dictionary, for, while)
    - リスト(`[]`)と辞書(`{}`)の操作
    - `for` 文による繰り返し処理
    - `while`, `break`, `continue` の制御
    - 成果物: 商品辞書を使った買い物アプリ
- Section 4: 関数とモジュール
    - 関数の定義 (`def`), 引数の初期値, `return`
    - モジュール (`import`) と標準ライブラリ (`random`)
    - 成果物: じゃんけんゲーム
- Section 5: 応用練習（アルゴリズム基礎）
    - リストと辞書の組み合わせ操作
    - 条件分岐(`if`)と繰り返し(`for`)を組み合わせた集計処理
    - 成果物: 条件付き合計金額計算プログラム（ECサイト風）
- Section 6: クラスとオブジェクト指向 
    - クラスの定義 (`class ClassName:`)
    - インスタンス生成 (`ClassName()`)
    - インスタンス変数の設定と参照


## 🛠 開発環境
- macOS
- Python 3.12.1
- VS Code
